# TestSmtpServer

メール送信機能の開発に利用できるテスト用の SMTP サーバ

## 主な機能

- SMTP に準拠したメールの受信と受信内容、送受信ログの確認
- STARTTLS による SSL 通信
- SMTP AUTH による認証
- MIME デコード
